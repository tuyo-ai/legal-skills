# LEGAL SKILLS 🌿

The ultimate directory for Legal Skills, Agents, and MCPs.

🌐 Website: [legalskills.sh](https://legalskills.sh)

## 🚀 Mission
To provide a centralized, categorized, and searchable repository of AI-driven legal tools, organized by jurisdiction to ensure local compliance and accuracy.

## 📂 Repository Structure
- `data/skills/`: Directory of AI legal skills.
- `data/agents/`: Directory of autonomous legal agents.
- `data/mcps/`: Directory of Model Context Protocol servers.
