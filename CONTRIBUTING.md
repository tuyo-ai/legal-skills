# Contributing to Legal Skills

Thank you for helping us build the ultimate legal intelligence directory!

## How to contribute

We use a Pull Request (PR) workflow. To add new content, follow these steps:

1. **Fork the repository** and create your branch from `main`.
2. **Create your content file** using the templates provided in the `templates/` folder.
3. **Place your file** in the correct directory:
   - `data/skills/` for new AI skills.
   - `data/agents/` for new autonomous agents.
   - `data/mcps/` for new Model Context Protocol servers.
4. **Commit your changes** with a clear message.
5. **Submit a Pull Request** describing your contribution.

## Guidelines
- **Categorization:** Ensure you specify the correct `Country` and `Language` in your file metadata.
- **Format:** Stick to the provided Markdown templates.
- **Quality:** Write clear, actionable descriptions.

You can find the templates in the `templates/` directory.
